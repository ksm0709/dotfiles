# Language and Style

언어별 선호 스택과 코드 스타일 기준입니다.

## Preferred Stack

- **Languages**: C++, Python, TypeScript
- **Editor**: `vim`

## Global Language Rule

- 대화/문서는 한국어 기본, 코드/주석은 영어 기본으로 작성합니다.

## Language-specific Style

### Python

- PEP8 준수
- `ruff` 포맷팅
- Google 스타일 가이드 준수

### C++

- `clangd`, `clang-tidy`, `clang-format` 사용
- Google 스타일 가이드 준수

### TypeScript

- ESLint와 Prettier 사용
