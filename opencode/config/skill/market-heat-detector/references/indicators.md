# 선행지표 상세 설명

## VIX Put-Call Ratio

### 개요
VIX Put-Call Ratio는 CBOE 변동성 지수(VIX)와 관련된 풋-콜 옵션 비율을 나타내는 지표입니다. 투자자들의 시장 심리와 공포 수준을 측정하는 데 사용됩니다.

### 계산 방법
```
VIX Put-Call Ratio = VIX 관련 풋 옵션 거래량 / VIX 관련 콜 옵션 거래량
```

### 해석 기준
- **1.0 이상**: 공포 우세 (투자자들이 하락을 두려워하여 풋 옵션 매수)
- **0.7-1.0**: 중립 상태
- **0.7 이하**: 극도 낙관 (투자자들이 상승을 기대하여 콜 옵션 매수)

### 특징
- **리드 타임**: 1-4주
- **신뢰도**: 중간-높음
- **역사적 성과**: 시장 천정 예측에 높은 정확도

### 사용 팁
- 0.5 이하는 극심한 과열 신호
- 1.5 이상은 극심한 공포 신호
- 5일 이동평균과 함께 분석하면 신뢰도 향상

## Advance/Decline Breadth (A/D Breadth)

### 개요
Advance/Decline Breadth는 시장의 내부 강도와 참여도를 측정하는 지표입니다. 상승하는 종목 수와 하락하는 종목 수의 비율을 분석합니다.

### 계산 방법
```
A/D Ratio = 상승 종목 수 / (상승 종목 수 + 하락 종목 수)
A/D Line = 누적(상승 종목 수 - 하락 종목 수)
```

### 해석 기준
- **0.6 이상**: 강세장 (시장 전반적으로 상승)
- **0.4-0.6**: 보합장
- **0.4 이하**: 약세장 (시장 전반적으로 하락)

### 다이버전스 분석
- **베어리시 다이버전스**: 가격 상승 but A/D 라인 하락 → 시장 천정 신호
- **불리시 다이버전스**: 가격 하락 but A/D 라인 상승 → 시장 바닥 신호

### 특징
- **리드 타임**: 2-6주
- **신뢰도**: 높음
- **민감도**: 중간

### 사용 팁
- 50일 이동평균과 함께 분석
- NYSE, NASDAQ 별도 분석 후 종합
- 거래량과 결합하면 신뢰도 향상

## New Highs/New Lows Ratio

### 개요
New Highs/New Lows Ratio는 52주 신고가와 신저가를 기록한 종목 수의 비율을 분석하여 시장의 내부 강도를 측정합니다.

### 계산 방법
```
New Highs/Lows Ratio = 신고가 종목 수 / (신고가 종목 수 + 신저가 종목 수)
```

### 해석 기준
- **0.7 이상**: 강세 확신 (다수 종목이 신고가 기록)
- **0.3-0.7**: 정상 범위
- **0.3 이하**: 약세 확신 (다수 종목이 신저가 기록)

### 특별 패턴
- **신고가 감소 + 신저가 증가**: 시장 천정 신호
- **신저가 감소 + 신고가 증가**: 시장 바닥 신호

### 특징
- **리드 타임**: 1-3주
- **신뢰도**: 중간
- **선행성**: 높음

### 사용 팁
- 10일 이동평균으로 노이즈 제거
- 거래소별(NYSE, NASDAQ) 분석
- 시가총액 가중 분석

## Sentiment Indicators

### AAII 투자자 심리 지표

#### 개요
American Association of Individual Investors(AAII)에서 주간으로 조사하는 개인 투자자 심리 지표입니다.

#### 계산 방법
```
Bullish Percentage = (낙관적 응답 수 / 전체 응답 수) × 100
Bearish Percentage = (비관적 응답 수 / 전체 응답 수) × 100
Bull-Bear Spread = Bullish % - Bearish %
```

#### 해석 기준
- **Bullish > 60%**: 극도 낙관 (과열 신호)
- **Bullish 20-40%**: 정상 범위
- **Bullish < 20%**: 극도 비관 (바닥 신호)

#### 특징
- **리드 타임**: 2-8주
- **신뢰도**: 중간-높음
- **역발성**: 높음 (역발성 지표)

### CNN Fear & Greed Index

#### 개요
CNN에서 제공하는 시장 심리 종합 지표로, 7개 지표를 조합하여 계산합니다.

#### 구성 지표
1. 주가 모멘텀 (S&P 500 vs 125일 이동평균)
2. 주가 강도 (신고가/신저가 비율)
3. 주가 너비 (A/D 브레드th)
4. 마진 디버트 (NYSE 마진 디버트)
5. 풋-콜 옵션 비율
6. 변동성 (VIX)
7. 안전 자선 수요 (채드 vs 주식)

#### 해석 기준
- **0-25**: 극심한 공포
- **25-45**: 공포
- **45-55**: 중립
- **55-75**: 탐욕
- **75-100**: 극심한 탐욕

- **CRITICAL**: 80점 이상

#### VIX Put-Call Ratio
- 과열: < 0.7
- 정상: 0.7-1.0
- 공포: > 1.0

#### A/D Breadth
- 강세: > 0.6
- 정상: 0.4-0.6
- 약세: < 0.4

#### New Highs/Lows Ratio
- 강세: > 0.7
- 정상: 0.3-0.7
- 약세: < 0.3

#### Sentiment (Bullish %)
- 과열: > 60%
- 정상: 20-40%
- 공포: < 20%

- 단기: 5-10일
- 중기: 10-20일

- **약함**: 추세 차이 < 10%

2. 추세 라인 기울기 차이 확인
3. 거래량 다이버전스 동반 여부
